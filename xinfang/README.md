# 警务管理系统

这是一个用于警务管理的网站系统，具有市级和县（区）级两个层级的权限和功能。

## 功能列表

### 市级功能
1. 信息登记录入
2. 信息编辑查询
3. 受理派发流转
4. 移交办理（督察/法制/纪检）
5. 督办计时
6. 接收反馈
7. 落实审批
8. 差件退回
9. 统计报表
10. 平台账号管理和权限分派

### 县（区）级或市局警种部门功能
1. 开始办理
2. 落办反馈
3. 审批结论
4. 退件处理
5. 超期情况查看

## 项目结构
- `index.html`: 网站首页及登录界面
- `css/`: 样式文件
- `js/`: JavaScript 脚本
- `pages/`: 网站各功能页面
- `images/`: 网站图片资源
- `uploads/`: 上传文件存放目录

## 技术栈
- HTML5
- CSS3
- JavaScript
- 本地存储
